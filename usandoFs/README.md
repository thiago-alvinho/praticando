# Usando Fs

Prática da utilização da biblioteca File System do Node.js construindo uma aplicação que recebe um arquivo texto e cria um arquivo resultado.txt que contém todas as palavras que possuem duas ou mais ocorrências em um mesmo parágrafo.

## Motivação

Criar uma aplicação que facilite a revisão de texto para retirar palavras repetidas em um mesmo parágrafo.
